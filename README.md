* 코딩 테스트 대비 [백준](https://www.acmicpc.net/step) 코딩테스트 문제풀이

* 윤성우의 열혈 자료구조 책을 통한 자료구조 학습
  * 2인 1개조 동료평가 진행(1주 2파트 진행)
  * 깃허브 개념 정리
* 개인과제 계획 및 실천 후 깃허브 기록
  * 깃허브 꾸준히 관리하기
  * 깃에 익숙해지기
